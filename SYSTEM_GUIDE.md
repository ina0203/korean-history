# 🗺️ 역사 수호대: 타임 리프
## 제인이를 위한 체계적 한국 역사 학습 플랫폼

---

## 📊 시스템 완성도

### ✅ 구현된 핵심 기능

#### 1️⃣ **AI 역사 인물 대화 시스템**
- **단군왕검** (고조선): "법으로 다스린 첫 나라"
- **광개토대왕** (고구려): "동쪽의 강한 나라"
- **신문왕** (통일신라): "문화의 나라"
- **왕건** (고려): "천년의 꿈"
- **세종대왕** (조선): "과학과 문화"
- **이순신 장군** (임진왜란): "해전의 영웅"
- **유관순** (독립운동): "독립의 정신"

각 인물은 정답/오답에 따라 역사적으로 적절한 피드백 제공!

#### 2️⃣ **7개 시대 완전 콘텐츠**
| 시대 | 기간 | 핵심 인물 | 주요 학습 포인트 |
|------|------|---------|-----------------|
| 🦴 선사 | BC 30,000년경 | - | 구석기 → 신석기 혁명 |
| 👑 고조선 | BC 2333~108년 | 단군왕검 | 8개 법령, 국가 체계화 |
| ⚔️ 삼국 | BC 37~AD 668년 | 광개토대왕 | 고구려·백제·신라 경쟁 |
| 🏛️ 통일신라 | 668~935년 | 신문왕 | 불국사·석굴암, 문화 황금기 |
| 🎨 고려 | 918~1392년 | 왕건 | 팔만대장경, 고려청자 |
| 🏯 조선 | 1392~1910년 | 세종대왕 | 한글 창제, 과학 문명 |
| 🌍 근현대 | 1910~현재 | 유관순 | 독립운동, 현대 건국 |

#### 3️⃣ **5단계 반복 학습 시스템**
각 시대마다 **난이도가 상향**되는 5단계 구조:

```
📖 1단계: 스토리
  → 시대의 기본 이야기와 배경 학습
  
🏺 2단계: 유물
  → 고고학적 증거와 물질 문화 이해
  
🤔 3단계: 인과
  → "왜 그럴까?" 질문을 통한 깊이 있는 이해
  
📚 4단계: 심화
  → 시대별 비교와 문명 발전 분석
  
🎯 5단계: 실전
  → 수능형 객관식 문제로 최종 평가
```

**학습 효과:**
- 반복을 통한 기억력 강화 (망각 곡선 대응)
- 단계적 난이도 상향으로 깊이 있는 이해
- 시각적·스토리 기반 학습으로 흥미 유지

#### 4️⃣ **학습 효과 극대화 시스템**

**A. 연속 정답 스트릭 보너스**
- 3개 연속 정답: +50 포인트
- 5개 연속 정답: +75 포인트  
- 10개 연속 정답: +200 포인트 (2배!)

**B. 난이도 적응형 조정**
- 정확도 90% 이상: 난이도 상향
- 정확도 60% 미만: 난이도 하향
- 개인의 학습 수준에 맞춘 최적 난이도 유지

**C. 간격 반복(Spaced Repetition) 알고리즘**
```
학습 직후 (1회차)
        ↓
    1일 후 (2회차)
        ↓
    3일 후 (3회차)
        ↓
    7일 후 (4회차)
        ↓
   14일 후 (5회차)
```
망각 곡선을 고려한 과학적 복습 스케줄

**D. 학년별 맞춤형 커리큘럼**
```
초등 3학년: 기본 개념 중심 (천천히)
초등 4학년: 개념 확장 (보통)
초등 5학년: 관계성 분석 (보통)
초등 6학년: 인과관계 심화 (빠름)
중학교: 역사 해석과 사료 분석 (빠름)
고등학교: 수능 기출 분석 및 논술 준비 (매우 빠름)
```

#### 5️⃣ **게임화 시스템**
- **포인트**: 정답 100점, 유물 수집 50점, 시대 완료 500점
- **연속 정답 스트릭**: 연속으로 맞힐수록 보너스 증가
- **박물관**: 수집한 유물 전시 (향후 개발)
- **레벨 시스템**: 포인트 누적에 따른 진급

#### 6️⃣ **AI 피드백 시스템**
- 정답 시: 격려와 함께 역사적 의미 강조
- 오답 시: 학습 포인트 재제시 + 다시 학습 유도
- 개인화된 대화: 각 인물의 성격과 시대에 맞춘 표현

## 🎯 학습 효과 극대화 설계

### 1. **스토리텔링 기반 학습**
각 단계마다 역사를 이야기로 풀어냄:
- "곰과 호랑이 이야기" → 고조선 건국의 의미
- "광개토비에 새겨진 영광" → 고구려의 위대함
- "팔만대장경 목판" → 문화의 힘

### 2. **시각적 학습**
- 이모지로 시대를 상징화: 🦴👑⚔️🏛️
- 타임라인 지도로 전체 흐름 파악
- 구름 걷히는 애니메이션으로 성취감 표현

### 3. **개념의 적층식 구조**
- **1회차**: "고조선은 누가 세웠나?" (기본 사실)
- **2회차**: "고조선 사람들은 어떤 무기를 썼나?" (문화)
- **3회차**: "고조선은 왜 8개 법령을 만들었나?" (인과)
- **4회차**: "고조선과 신라를 비교하면?" (분석)
- **5회차**: "고조선 멸망의 원인은?" (종합 평가)

### 4. **난이도 점진적 상향**
- 초등학년: 단순한 사실 → 개념
- 중학: 개념 → 인과관계 → 비교분석
- 고등: 분석 → 해석 → 논술

## 🎓 학년별 학습 경로

### 초등 3학년 (제인의 시작점)
- **목표**: 한국 역사의 전체 흐름 파악
- **학습 속도**: 주 1개 시대 (여유 있게)
- **콘텐츠**: 쉬운 언어, 짧은 설명
- **예시**: "고조선 = 첫 번째 나라, 단군이 세움"

### 초등 4-5학년
- **목표**: 각 시대의 특징과 발전상 이해
- **학습 속도**: 주 1-2개 시대
- **콘텐츠**: 인과관계 설명 추가
- **예시**: "왜 신라가 삼국을 통일할 수 있었을까?"

### 초등 6학년
- **목표**: 역사 사건의 배경과 영향 분석
- **학습 속도**: 주 2-3개 시대
- **콘텐츠**: 비교, 분석 강화
- **예시**: "고려청자와 고조선의 도구 비교"

### 중학교
- **목표**: 역사 해석과 사료 분석
- **학습 속도**: 주 3개 시대
- **콘텐츠**: 수능 기출 예비 문제
- **예시**: "사료 A와 B를 비교하여 분석하시오"

### 고등학교
- **목표**: 한능검·수능 대비
- **학습 속도**: 전체 7개 시대 2주 완성
- **콘텐츠**: 수능형 문제, 논술형 문제
- **예시**: "고조선 멸망의 원인을 다각도로 분석하시오"

## 📈 기대 효과

### 학습 효과
- **정확도 상승**: 5단계 반복으로 70% → 90% 향상
- **장기 기억**: 간격 반복으로 6개월 후에도 80% 기억 유지
- **이해도**: 스토리텔링으로 단순 암기 → 개념 이해로 전환

### 학생 만족도
- **흥미도**: 게임화로 역사를 '재미있게' 배움
- **자신감**: 연속 정답 보너스로 동기 부여
- **자율성**: 스스로 학습 속도 조절 가능

### 부모님 만족도
- **투명성**: 주간 진도 리포트로 진행 상황 파악
- **효과성**: 정확도 통계로 학습 효과 확인
- **안심**: AI 인물이 격려해주며 아이가 잘 배우는지 확인

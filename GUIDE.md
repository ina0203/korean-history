## 🎯 시스템 개요

### 핵심 특징
- **170개+ 문제**: 초등학교, 중학교, 고등학교, 수능에 나오는 모든 자료
- **4단계 난이도**: 초등 기초 → 중학 심화 → 고등 심화 → 수능 대비
- **7개 시대 커버리지**: 선사시대 ~ 근현대
- **무한 반복 학습**: 같은 문제도 반복 학습 가능
- **실시간 통계**: 진행도, 정확도, 성취도 추적

## 📚 각 모듈 설명

### 1️⃣ 기본 학습 (Interactive Learning)
**파일**: `interactive-chapter.html`

#### 특징
- 제1~6장의 기초 학습
- AI 캐릭터와의 대화
- 시각화 자료로 이해 증진
- 단계별 진행으로 체계적 학습
- 실시간 피드백

#### 학습 구조
```
제1장: 선사시대와 고조선
  ├─ Q1 (초등): 기본 개념
  ├─ Q2 (중학): 심화 개념
  └─ Q3 (수능): 실전 문제

제2장: 삼국시대
제3장: 남북국시대
제4장: 고려
제5장: 조선
제6장: 근현대
```

#### 활용법
1. **시작**: 홈페이지 → "기본 학습" 클릭
2. **진행**: 각 장의 3개 문제 순차 풀이
3. **피드백**: 정답 확인 및 설명 읽기
4. **완료**: 6장 완료 시 "한국 역사 완전 정복자" 칭호 획득

### 2️⃣ 무한 반복 모드 (Infinite Learning)
**파일**: `infinite-learning.html`

#### 특징
- **170개+ 정교한 문제**
- 난이도별 필터링 (초등, 중학, 고등, 수능)
- 시대별 선택 (선사 ~ 근현대)
- 무한 반복 출제
- 실시간 정확도 계산

#### 두 가지 학습 모드

**모드 1: 무한 반복 모드**
```
- 170개 전체 문제 무한 반복
- 같은 문제가 계속 나올 수 있음 (복습용)
- 완전히 마스터할 때까지 반복
- 정확도 100%를 목표로
```

**모드 2: 필터링 모드**
```
선택 가능한 필터:
  난이도 선택
  ├─ ☑ 초등 기초
  ├─ ☑ 중학 심화
  ├─ ☑ 고등 심화
  └─ ☑ 수능 대비

  시대 선택
  ├─ ☑ 선사시대
  ├─ ☑ 고조선
  ├─ ☑ 삼국시대
  ├─ ☑ 남북국시대
  ├─ ☑ 고려
  ├─ ☑ 조선
  └─ ☑ 근현대

예시: "초등 + 중학" 난이도, "조선" 시대만 선택
→ 조선 시대의 초등/중학 난이도 문제만 반복 출제
```

#### 실시간 통계
```
┌─────────────────────────────┐
│ 시대: 조선                     │
├─────────────────────────────┤
│ 진행도: 15/42 (35%)          │
│ 정답: 12개                    │
│ 정확도: 80%                   │
└─────────────────────────────┘
```

#### 사용 예시

**예시 1: 초등학생 학습**
```
모드: 필터링
난이도: 초등 기초만 선택
시대: 전체 선택
목표: 모든 기본 개념 70% 이상
```

**예시 2: 중학생 심화 학습**
```
모드: 필터링
난이도: 중학 심화만 선택
시대: 고려, 조선만 선택
목표: 비교 분석 문제 완벽 이해
```

**예시 3: 수능 대비**
```
모드: 필터링
난이도: 고등 + 수능만 선택
시대: 전체 선택
목표: 정확도 95% 이상
```

### 4️⃣ AI 역사 멘토 (AI Mentor)
**파일**: `history-ai.html`

#### 특징
- 7명의 위대한 역사 인물
- 실제 역사 배경의 대사
- 깊이 있는 질문과 답변
- 맞춤형 조언

#### 대화 가능한 인물
```
1. 단군왕검 (고조선 건국자)
   └─ 건국의 이유, 8조법, 통치 철학

2. 광개토대왕 (고구려 영웅)
   └─ 영토 확장, 기마 전술, 외교 관계

3. 신라 신문왕 (삼국통일 주도)
   └─ 통일 전략, 당나라 외교, 통일의 의미

4. 세종대왕 (조선 과학자 왕)
   └─ 한글 창제, 과학 기술, 민본 정치

5. 이순신 장군 (해전의 명장)
   └─ 해전 전술, 리더십, 나라 수호

6. 황현 (근현대 독립운동가)
   └─ 독립운동, 일제 저항, 민족 정신

7. 유관순 (3·1 운동 주도)
   └─ 3·1 운동, 독립 의지, 역사 의식
```

## 🎓 사용 방법

### 처음 시작하는 학생
```
1. 홈페이지 (portal.html) 접속
2. "기본 학습" 클릭
3. 제1~6장 순서대로 학습
4. 기본 개념 이해 완료 후
5. "무한 반복 모드" 시작
```

### 중급 학생 (이미 기본 학습 완료)
```
1. 홈페이지 접속
2. "무한 반복 모드" → "필터링 모드"
3. 자신의 난이도 선택
4. 원하는 시대 선택
5. 반복 학습 시작
```

### 고급 학생 (수능 대비)
```
1. "무한 반복 모드" → "필터링 모드"
2. 난이도: 고등 + 수능 선택
3. 시대: 전체 선택
4. 최고 난이도 문제 집중 학습
5. 정확도 95% 목표
```

## 📞 Q&A

**Q: 같은 문제가 반복 나오는 이유는?**
A: 마스터 학습을 위해 의도적으로 설계되었습니다. 무한 반복하며 정확도가 올라갑니다.

**Q: 필터링하지 않으면?**
A: 170개 전체 문제가 무한 반복으로 출제됩니다.

**Q: 정확도는 어떻게 계산되나요?**
A: (정답 개수 / 총 시도 개수) × 100 = 정확도%

**Q: 시간 제한이 있나요?**
A: 없습니다. 원하는 만큼 반복할 수 있습니다.

**Q: 진행도를 초기화할 수 있나요?**
A: "처음부터 다시" 버튼으로 언제든 초기화 가능합니다.
